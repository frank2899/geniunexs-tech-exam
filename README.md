## Getting Started

First, install the packages:

```bash
yarn install
```

First, run the development server:

```bash
yarn dev
```

## Commands

1. Prettier lint Check:

```bash
yarn format:check
```

2. Prettier Fix lint:

```bash
yarn format:write
```

3. Cypress e2e test:

```bash
yarn test:e2e
```

4. Cypress e2e UI:

```bash
yarn open:e2e
```

5. Test Units:

```bash

yarn test:unit
```
